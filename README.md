# SalaryApi
This serves as a calculation service for [money-ng-app frontend](https://github.com/Anteklantek/SalaryFrontend)

## Running
```
mvn clean package
cd target
java -jar SalaryCalculator-0.0.1-SNAPSHOT.jar
```
